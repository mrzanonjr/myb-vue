# myb-vue
Study project for initial steps in Vue.JS


## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Install dependency json-server
```
npm install -g json-server
```

### Running json-server
```
npm run backend
```

### Compiles and minifies for production
```
npm run build
```
